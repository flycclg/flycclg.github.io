docker命令别名

进入 .bashrc 中修改配置（ .bashrc 是 root 下的隐藏文件）

cd /root

vim .bashrc

在 .bashrc 中加入以下内容

alias dps='docker ps --format "table {{.ID}}\t {{.Image}}\t{{.Ports}}\t{{.Status}}\t{{.Names}}"'alias dis='docker images'

重新加载文件

source /root/.bashrc
